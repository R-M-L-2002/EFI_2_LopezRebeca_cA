Bienvenido!!

Este sistema permite la gestión integral de un local de venta de celulares. A continuación, se presenta un resumen de las funcionalidades disponibles:

Resumen de Modelos

Marca

Representa las marcas de equipos. Cada marca tiene un id único y un nombre. Está relacionada con Equipo.

Fabricante

Representa los fabricantes de modelos. Cada fabricante tiene un id, nombre, y pais_origen. Está relacionado con Modelo.

Modelo

Representa los modelos de equipos. Cada modelo tiene un id, nombre, y una relación con Fabricante. Está asociado con Equipo, Caracteristica, y Accesorio.

Categoria

Representa las categorías de equipos. Cada categoría tiene un id y un nombre. Está relacionada con Equipo.

Caracteristica

Describe características específicas de los modelos. Cada característica tiene un id, tipo, descripcion, y está asociada a Modelo. Está relacionada con Equipo.

Stock

Representa el inventario de equipos. Cada stock tiene un id, cantidad, ubicacion, y está asociado con Equipo. Incluye un estado de disponibilidad.

Proveedor

Representa los proveedores de inventarios. Cada proveedor tiene un id, nombre, y contacto. Está relacionado con Inventario.

Accesorio

Describe accesorios para modelos. Cada accesorio tiene un id, tipo, y está asociado con Modelo. Está relacionado con Equipo.

Equipo

Representa los equipos disponibles. Cada equipo tiene un id, nombre, precio, y está asociado con Categoria, Marca, Modelo, Caracteristica, y opcionalmente Accesorio. Está relacionado con Stock.

Inventario

Representa el inventario gestionado por proveedores. Cada inventario tiene un id, nombre, cantidad, fecha, y está asociado con Proveedor.

Cliente

Representa a los clientes que realizan compras. Cada cliente tiene un id y nombre. Está relacionado con Venta.

Venta

Registra las ventas realizadas. Cada venta tiene un id, cliente_id, equipo_id, precio, y fecha. Está asociado con Equipo.
